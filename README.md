# Game1

Early-stage 2D top-down survival game, built in Godot 4.

Very much a work in progress / me messing around and learning Godot properly. Loose plan is a Stardew-ish top-down world with day/night cycles, seasons, and corrupted nature enemies that come out at night for base defense waves.

## Status
Basic movement, tilemap, and day/night cycle in place. Nothing polished yet - mostly proof of concept stuff while I learn the engine.
